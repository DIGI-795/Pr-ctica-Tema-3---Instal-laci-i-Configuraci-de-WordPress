# Pràctica Tema 3 - Instal·lació i Configuració de WordPress

## Descripció d'aquest proyecte

* Aquest projecte tractaba sobre fer un lloc web amb Wordpress, jo he desarrollat una web per als fanàtics dels esports amb notícies dedicades als esports que consta de dos categoríes esports de pilota i arts marcials. El meu objectiu era mostrar els resultats i notícies del món dels esports en el meu lloc web funcional.

## Configuració del lloc

Elements importants:

* El menú principal de navegació
* La pàgina d'inici
    * Sublement - Qui som
* La creació i la publicació d'entrades
* La gestió del contingut des del panell d'administració de WordPress

Categoríes:

* Esports de pilota
* Arts marcials
